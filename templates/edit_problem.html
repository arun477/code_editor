Initial Code

This is what the user will be shown before starting their problems. The system will pass the Solution class to the Validation class during execution.

Validation Code

This is what the user will be shown before starting their problems. Solution and Validation classes are a must. The system will pass the Solution class to the Validation class during execution. The Validation class should return a tuple. The first element of the tuple should be a boolean indicating whether the test case passed or not, and the second element should be the return value, if any.
